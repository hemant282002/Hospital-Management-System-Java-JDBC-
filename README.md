# Hospital-Management-System-Java-JDBC-
## Hospital Management System (Java + JDBC)

This project is a **console-based Hospital Management System** developed using **Java and JDBC** to perform basic hospital database operations. The system allows users to manage patient records, doctor assignments, ward details, and hospital fees using a relational database.

The application connects to a database and performs various **CRUD operations (Create, Read, Update, Delete)** through SQL queries. It helps in storing and retrieving patient information efficiently.

### Key Features

* Create hospital database table
* Add new patient records
* Search patients by doctor name
* View patient details
* Update doctor assigned to a patient
* List patients with the same disease
* Apply discount on patient fees
* View all patient records
* Display patients in private or general wards
* Discharge patients (delete records)

### Technologies Used

* Java
* JDBC (Java Database Connectivity)
* SQL / PostgreSQL or MySQL
* Prepared Statements
* Object-Oriented Programming (OOP)

### Purpose

This project was developed to demonstrate how **Java applications interact with databases using JDBC** and how hospital data can be managed through structured queries.
